

# OIBSIP-Task-2

This repository contains the solution for **Task 2** of the Oasis Infobyte Internship Program. The task is to create a **Temperature Converter** using Python with a Graphical User Interface (GUI).

## Project Description

The application allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin. It is built using Python’s `tkinter` library and provides a user-friendly interface for entering values, selecting units, and viewing converted results.

## Features

- Convert temperatures between:
  - Celsius to Fahrenheit and Kelvin
  - Fahrenheit to Celsius and Kelvin
  - Kelvin to Celsius and Fahrenheit
- Clean and simple GUI interface
- Real-time conversion upon user interaction
- Input validation to ensure only numerical values are processed

## Technologies Used

- Python 3
- Tkinter (Python's built-in GUI library)

## Prerequisites

- Python 3.x must be installed on your system

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/sneha280904/OIBSIP-Task2.git
   ```

2. Navigate into the project directory:
   ```bash
   cd OIBSIP-Task2
   ```

3. Run the script:
   ```bash
   python temperature_converter.py
   ```
